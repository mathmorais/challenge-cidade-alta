### ⚡ Como iniciar

Instale as dependencias com o seu gerenciador de pacotes

Como **npm**

```bash
 npm install
```

Ou **yarn**

```bash
 yarn install
```

E inicialize o projeto em ambiente de desenvolvimento atravéz do comando:

```bash
 yarn dev
```
